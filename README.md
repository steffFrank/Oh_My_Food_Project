<!-- GETTING STARTED -->
## Getting Started  
To get a local copy up and running follow these simple example steps.

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/steffFrank/Oh_My_Food_Project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
